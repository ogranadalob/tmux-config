# Tmux Config

This is a simple tmux config.

## Installation 

Paste next code in your terminal:
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ogranadalob/tmux-config/HEAD/install.sh)"
```
Then open TMUX and execute `<leader>I`.

### Note
Is strongly suggested to install fira code from [nerd fonts](https://www.nerdfonts.com/font-downloads).
